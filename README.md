# 📊 Trader Behavior Insights using Market Sentiment

## 🔍 Objective
Analyze how Bitcoin market sentiment (Fear vs Greed) impacts trader performance and behavior.

## 📂 Datasets
- Historical Trader Data
- Fear & Greed Index

## ⚙️ Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Google Colab

## 📊 Key Analysis
- Profitability vs Sentiment
- Trade Activity Analysis
- Buy/Sell Behavior
- Risk Trends

## 🧠 Key Insights
- Higher profits observed during Greed phases
- Traders take higher risks during bullish markets
- Fear phases show reduced trading activity
- Sell trades dominate during Fear periods

## 🚀 Strategy Recommendations
- Use trend-following strategies in Greed
- Reduce leverage in volatile conditions
- Focus on risk management during Fear

## 📌 Conclusion
Market sentiment plays a crucial role in trader behavior and profitability. Understanding these patterns can improve trading strategies.
